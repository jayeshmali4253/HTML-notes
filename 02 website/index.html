Frontend Development

Frontend developent is the development of the graphical user interface of a website,through the use of HTML , CSS and Javascript so that the user can view and interact with that website

  • HTML: The Hypertext Markup Language or HTML is standard markup language for designed to be displayed in a web browser.


  • CSS : Cascading Style Sheets (CSS) is a style sheet language for describing the presentation of document written in a markup language such as HTML or XMLbr

    1. Pure CSS

    2. Bootstrap (Framework)

    3. Tailwind (Framework)

    4. Sass (Preprocessor)


  • Javascript: Javascript often abbreviated JS, is a progrramming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. it is use to add functionality in the website

    1. vanilla JS

    2. jQuery (Library)

    3. React (Framework)

    4. Angular (Framework)

    5. Vue (Framework)

    6. Typescript (Preprocessor)
